# AI-Assisted Alluvial Gold Prospectivity Mapping

## Overview

A QGIS-based geospatial workflow demonstrating georeferencing, coordinate transformations, DEM processing, drainage extraction, spatial analysis, GeoAI-assisted feature extraction, and workflow automation using PyQGIS and GeoAgent.

## Skills Demonstrated

- QGIS
- PyQGIS
- Georeferencing
- Coordinate Systems & Projections
- Raster Analysis
- Vector Analysis
- Geoprocessing
- DEM Analysis
- Spatial Joins
- Buffer Analysis
- GeoAI
- Workflow Automation
