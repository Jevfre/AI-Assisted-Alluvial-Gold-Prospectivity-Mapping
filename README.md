# AI-Assisted Alluvial Gold Prospectivity Mapping

## Project Overview

This repository demonstrates a QGIS-based geospatial workflow for alluvial gold prospectivity mapping using remote sensing, DEM-derived terrain analysis, GeoAI-assisted feature extraction, and PyQGIS workflow automation.

The project was designed to demonstrate modern GIS methodologies for identifying geomorphological features commonly associated with placer mineral accumulation while maintaining full confidentiality of operational exploration data.

---

## Objectives

- Standardize geospatial datasets from multiple sources
- Perform coordinate transformation and CRS validation
- Extract hydrological features from DEM data
- Apply raster and vector analysis workflows
- Automate repetitive GIS tasks using PyQGIS
- Demonstrate GeoAI-assisted feature extraction workflows
- Produce exploration-ready cartographic outputs
