# GeoAgent Workflow Prompts

## Data Preparation

Generate drainage network from DEM.

Validate coordinate reference systems for all imported datasets.

Identify datasets requiring reprojection to the Nigeria Transverse Mercator coordinate system.

## Hydrological Analysis

Calculate flow accumulation raster.

Identify major drainage channels.

Delineate catchments from DEM-derived drainage networks.

## Exploration Targeting

Buffer historical artisanal mining occurrences by 500m.

Identify catchments intersecting favorable geological units.

Rank prospectivity zones based on drainage density and historical workings.

## Deliverables

Generate prospectivity map.

Export exploration target layers.

Create cartographic layout for reporting.
