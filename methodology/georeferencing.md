# Georeferencing Workflow

## Purpose

Historical geological maps and artisanal mining reports are often available as scanned images without spatial reference information.

## Tool Used

- QGIS Georeferencer

## Workflow

1. Import scanned geological map.
2. Select ground control points (GCPs).
3. Assign Coordinate Reference System (CRS).
4. Apply transformation.
5. Validate spatial accuracy.
6. Overlay with Sentinel-2 imagery and other exploration datasets.

## Output

Georeferenced datasets aligned with modern satellite imagery and exploration layers for subsequent spatial analysis.
