# Data Acquisition

## Objective

Acquire and prepare geospatial datasets required for AI-assisted prospectivity mapping and spatial analysis workflows.

## Data Sources

### Satellite Imagery

- Sentinel-2 multispectral imagery
- Used for visual interpretation and GeoAI-assisted feature extraction

### Elevation Data

- SRTM Digital Elevation Model (DEM)
- Used for terrain analysis, slope generation, drainage extraction, and hydrological modelling

### Historical References

- Published geological maps
- Historical mining records
- Public geological reports

### Vector Datasets

- Drainage networks
- Administrative boundaries
- Geological contacts
- Field observation points

## Data Preparation Workflow

1. Download and organize source datasets.
2. Verify spatial reference systems.
3. Standardize coordinate systems.
4. Inspect data quality.
5. Remove duplicate and invalid features.
6. Prepare layers for raster and vector analysis.

## Deliverables

- Standardized geospatial database
- Clean raster datasets
- Validated vector layers
- Analysis-ready project workspace
