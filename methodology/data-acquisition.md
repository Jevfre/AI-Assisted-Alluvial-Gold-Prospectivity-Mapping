# AI-Assisted Alluvial Gold Prospectivity Mapping

## Overview

This repository demonstrates a QGIS-based workflow for alluvial gold exploration using remote sensing, terrain analysis, GeoAI-assisted feature extraction, and GeoAgent workflow automation.

The objective is to reduce exploration uncertainty by integrating geological evidence, historical artisanal mining activity, hydrological analysis, and satellite imagery into a unified prospectivity mapping workflow.
