# Catchment Analysis

## Input Data

- SRTM DEM
- Sentinel-2 Imagery

## QGIS Tools Used

- Slope
- Hillshade
- Raster Calculator
- Flow Direction
- Flow Accumulation

## Workflow

1. Import DEM into QGIS.
2. Generate slope and hillshade layers.
3. Calculate flow direction.
4. Generate flow accumulation raster.
5. Delineate drainage networks and catchments.
6. Identify areas likely to concentrate alluvial sediments.

## Exploration Objective

Identify drainage systems and depositional environments favorable for alluvial gold accumulation.
